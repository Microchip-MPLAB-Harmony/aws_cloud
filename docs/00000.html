Microchip Support
Introduction

The Amazon FreeRTOS is abstracted as Harmony 3 component to easily configure and generate code to develop cloud connected applications using Harmony 3 framework. It uses updated version of Amazon FreeRTOS 201908.00 to support Microchip products for code generation. 

 

The Amazon FreeRTOS is supported in the following products.

  • SAM E70/S70/V70/V71 Family
  • SAM D5x/E5x Family
  • PIC32MZ Embedded Connectivity with Floating Point Unit (EF) Family

 

This repository contains following three components for configuration and code generation.

  1. "AmazonFreeRTOS" Component

Amazon FreeRTOS component is used to configure and generate Amazon FreeRTOS code 

 

  1. AmazonHWInterface component

The Hardware interface components is used for device specific configuration and code generation 

 

  1. AmazonDeviceTester component

Amazon DeviceTester component is used to configure and generate code for qualification testing. It adds required dependencies for testing the Amazon FreeRTOS solution using Amazon IDT (refer: https://docs.aws.amazon.com/freertos/latest/userguide/device-tester-for-freertos-ug.html).
