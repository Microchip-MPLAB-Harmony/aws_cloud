LED Shadow Client Application

This example application shows how to use the shadow client feature to set/get Device status using LED(s). The device turns LED ON/OFF once to indicate the Shadow state is ready following which the user may turn LED(s) ON/OFF from AWS console. The application uses ECC608 for key storage and authentication.
